ascii-chat(1) User Commands ascii-chat(1)

NAME

ascii-chat - Video chat in your terminal

SYNOPSIS

ascii-chat [options] [server | client | mirror | discovery-service] [mode-options]

ascii-chat [options] <session-string>

USAGE

ascii-chat [options...]
Start a new session (share the session string)
ascii-chat <session-string> [options...]
Join an existing session
ascii-chat <mode> [options...]
Run in a specific mode
ascii-chat server [bind-address] [bind-address] [mode-options...]
Start server (can specify 0-2 bind addresses for IPv4/IPv6)
ascii-chat client [address] [mode-options...]
Connect to server (defaults to localhost:27224)
ascii-chat mirror [mode-options...]
View local webcam or media file as ASCII art
ascii-chat discovery-service [bind-address] [bind-address] [mode-options...]
Start discovery service (can specify 0-2 bind addresses for IPv4/IPv6)

EXAMPLES

ascii-chat
Start new session (share the session string)
ascii-chat ninety-aunty-velocity
Join session with session string
ascii-chat spoilt-seek-precinct
Join a session using the session string
ascii-chat dummy-theta-blush --color
Join session with color support
ascii-chat ironic-nightclub-bit --discovery-server discovery.example.com
Join session via custom discovery server
ascii-chat wan-teeth-intro -f video.mp4
Join session and stream from local video file
ascii-chat five-noodle-doorstep --url 'https://www.youtube.com/watch?v=tQSbms5MDvY'
Join session and stream from YouTube video
ascii-chat amateur-paragraph-daytime -f '-'
Join session and stream media from stdin (cat file.mov | ascii-chat ... -f '-')
ascii-chat bitty-dryer-trust --palette-chars '@%#*+=-:. '
Join session with custom ASCII palette characters
ascii-chat server
Run as dedicated server
ascii-chat server
Start on localhost (127.0.0.1 and ::1)
ascii-chat server 0.0.0.0
Start on all IPv4 interfaces
ascii-chat server 0.0.0.0 ::
Start on all IPv4 and IPv6 interfaces (dual-stack)
ascii-chat server --port 8080
Start on custom port
ascii-chat server --key ~/.ssh/id_ed25519 --discovery
Start with identity key and discovery registration
ascii-chat client example.com
Connect to specific server
ascii-chat client
Connect to localhost
ascii-chat client example.com
Connect to remote server
ascii-chat client example.com:8080
Connect to remote server on custom port
ascii-chat client --url 'https://www.youtube.com/watch?v=tQSbms5MDvY'
Stream from YouTube URL
ascii-chat client -f video.mp4
Stream from local video file
ascii-chat client --color-mode mono --render-mode half-block --width 120
Connect with custom display options
ascii-chat client --palette-chars '@%#*+=-:. '
Use custom ASCII palette characters
ascii-chat client --snapshot
Capture single frame and exit
ascii-chat mirror
Preview local webcam as ASCII
ascii-chat mirror
View local webcam
ascii-chat mirror --color-mode mono
View webcam in black and white
ascii-chat mirror --url 'https://www.youtube.com/watch?v=tQSbms5MDvY'
Stream from YouTube URL
ascii-chat mirror -f video.mp4
Stream from local video file (supports mp4, mkv, webm, mov, etc)
ascii-chat mirror -f '-'
Stream media from stdin (cat file.gif | ascii-chat mirror -f '-')
ascii-chat mirror --palette-chars '@%#*+=-:. '
View with custom ASCII palette characters
ascii-chat mirror --snapshot
Capture single frame and exit
ascii-chat discovery-service
Start on localhost
ascii-chat discovery-service 0.0.0.0
Start on all IPv4 interfaces
ascii-chat discovery-service --port 5000
Start on custom port
ascii-chat discovery-service --require-server-identity --require-client-identity
Enforce identity verification for all parties

OPTIONS

LOGGING

-L, --log-file STR
Set FILE as path for log file (mode: global) (env: ASCII_CHAT_LOG_FILE)
--log-level VAL
Set log level: dev, debug, info, warn, error, fatal (mode: global) (env: ASCII_CHAT_LOG_LEVEL)
-V, --verbose VAL
Increase log verbosity (stackable: -VV, -VVV) (mode: global) (env: ASCII_CHAT_VERBOSE)
-q, --quiet
Disable console logging (log to file only) (default: false) (mode: global) (env: ASCII_CHAT_QUIET)

CONFIGURATION

--config STR
Load configuration from toml FILE (mode: global)
--config-create
Create default config file and exit (optionally specify output path) (default: false) (mode: global)

SHELL

--completions STR
Generate shell completions (bash, fish, zsh, powershell) (mode: global)
--create-man-page
Create man page for ascii-chat (default: false) (mode: global)

TERMINAL

-x, --width NUM
Terminal width in characters. Can be controlled using $COLUMNS. (default: 110) (modes: client, mirror) (env: ASCII_CHAT_WIDTH)
-y, --height NUM
Terminal height in characters. Can be controlled using $ROWS. (default: 70) (modes: client, mirror) (env: ASCII_CHAT_HEIGHT)

WEBCAM

-c, --webcam-index NUM
Webcam device index to use for video input (default: 0) (modes: client, mirror) (env: ASCII_CHAT_WEBCAM_INDEX)
-g, --webcam-flip
Flip webcam output horizontally before using it (default: true) (modes: client, mirror) (env: ASCII_CHAT_WEBCAM_FLIP)
--test-pattern
Use test pattern instead of webcam (default: false) (modes: client, mirror) (env: WEBCAM_DISABLED)
--list-webcams
List available webcam devices and exit (modes: client, mirror)

AUDIO

--list-microphones
List available audio input devices and exit (modes: client, mirror)
--list-speakers
List available audio output devices and exit (modes: client, mirror)
-A, --audio
Enable audio streaming (default: false) (modes: client) (env: ASCII_CHAT_AUDIO)
--microphone-index NUM
Microphone device index for audio input (default: -1) (modes: client) (env: ASCII_CHAT_MICROPHONE_INDEX)
--speakers-index NUM
Speakers device index to use for audio output (default: -1) (modes: client) (env: ASCII_CHAT_SPEAKERS_INDEX)
--microphone-sensitivity NUM
Microphone volume multiplier (0.0-1.0) (default: 1.00) (modes: client) (env: ASCII_CHAT_MICROPHONE_SENSITIVITY)
--speakers-volume NUM
Speaker volume multiplier (0.0-1.0) (default: 1.00) (modes: client) (env: ASCII_CHAT_SPEAKERS_VOLUME)
--audio-analysis
Enable audio analysis (debug) (default: false) (modes: client) (env: ASCII_CHAT_AUDIO_ANALYSIS)
--no-audio-playback
Disable speakers output (default: false) (modes: client) (env: ASCII_CHAT_NO_AUDIO_PLAYBACK)
--encode-audio
Enable Opus audio encoding (default: true) (modes: client) (env: ASCII_CHAT_ENCODE_AUDIO)
--no-encode-audio
Disable Opus audio encoding (default: false) (modes: client) (env: ASCII_CHAT_NO_ENCODE_AUDIO)

DISPLAY

--color-mode VAL
Terminal color level (auto, none, 16, 256, truecolor). This controls what ANSI escape codes ascii-chat will use. (modes: client, mirror) (env: ASCII_CHAT_COLOR_MODE)
-M, --render-mode VAL
ascii render mode (foreground, background, half-block) (modes: client, mirror) (env: ASCII_CHAT_RENDER_MODE)
-P, --palette VAL
ascii palette type (standard, blocks, digital, minimal, cool, custom). All but custom are built-in presets that look nice. Try them out! (modes: client, mirror) (env: ASCII_CHAT_PALETTE)
-C, --palette-chars VAL
Custom palette characters (implies --palette=custom) for rendering images to ascii. These characters only will be used to create the rendered output. Can be UTF-8 content (see --utf8). (modes: client, mirror) (env: ASCII_CHAT_PALETTE_CHARS)
--show-capabilities
Show detected terminal capabilities and exit (default: false) (modes: client, mirror) (env: ASCII_CHAT_SHOW_CAPABILITIES)
--utf8
Force UTF-8 support. By default UTF-8 is automatically detected and enabled if the terminal supports it. (default: false) (modes: client, mirror) (env: ASCII_CHAT_UTF8)
-s, --stretch
Allow aspect ratio distortion of image for rendering ascii output. This can allow the rendered ascii to fill your terminal. (default: false) (modes: client, mirror) (env: ASCII_CHAT_STRETCH)
--strip-ansi
Strip ANSI escape sequences from output before printing. Useful for scripting and debugging. (default: false) (modes: client, mirror) (env: ASCII_CHAT_STRIP_ANSI)
--fps NUM
Target framerate for rendering ascii (1-144, 0=use default). (default: 0) (modes: client, mirror) (env: ASCII_CHAT_FPS)

SNAPSHOT

-S, --snapshot
Snapshot mode (one frame and exit) (default: false) (modes: client, mirror) (env: ASCII_CHAT_SNAPSHOT)
-D, --snapshot-delay NUM
Snapshot delay in seconds. The timer starts right before the client-side program prints the first frame. --snapshot --snapshot-delay=0 will print the first frame and exit. (default: 4.00) (modes: client, mirror) (env: ASCII_CHAT_SNAPSHOT_DELAY)

PERFORMANCE

--compression-level NUM
zstd compression level (1-9) (default: 3) (modes: client, server) (env: ASCII_CHAT_COMPRESSION_LEVEL)
--no-compress
Disable compression (default: false) (modes: client, server) (env: ASCII_CHAT_NO_COMPRESS)

SECURITY

-E, --encrypt
Enable end-to-end encryption (requires the other party to be encrypted as well) (default: true) (modes: client, server, discovery-service) (env: ASCII_CHAT_ENCRYPT)
-K, --key STR
Server identity key (SSH Ed25519 or GPG key file, gpg:FINGERPRINT, github:USER, gitlab:USER, or HTTPS URL like https://example.com/key.pub) (modes: client, server, discovery-service) (env: ASCII_CHAT_KEY)
--password STR
Shared password for authentication (modes: client, server, discovery-service) (env: ASCII_CHAT_PASSWORD)
--no-encrypt
Disable encryption (requires the other party to be unencrypted as well) (default: false) (modes: client, server, discovery-service) (env: ASCII_CHAT_NO_ENCRYPT)
--server-key STR
Expected server public key for verification (SSH Ed25519 or GPG key file, gpg:FINGERPRINT, github:USER, gitlab:USER, or HTTPS URL like https://example.com/key.pub) (modes: client) (env: ASCII_CHAT_SERVER_KEY)
--client-keys STR
Allowed client keys (comma-separated: file paths with one key per line, github:USER, gitlab:USER, gpg:KEYID, or HTTPS URLs) (modes: server, discovery-service) (env: ASCII_CHAT_CLIENT_KEYS)
--discovery-insecure
Skip server key verification (MITM-vulnerable, requires explicit opt-in) (default: false) (modes: client) (env: ASCII_CHAT_DISCOVERY_INSECURE)
--discovery-server-key STR
Discovery server public key for verification (SSH Ed25519 or GPG key file, gpg:FINGERPRINT, github:USER, gitlab:USER, or HTTPS URL like https://discovery.ascii-chat.com/key.pub) (modes: client) (env: ASCII_CHAT_DISCOVERY_SERVER_KEY)
--require-server-identity
Require servers to provide signed Ed25519 identity (default: false) (modes: discovery-service) (env: ASCII_CHAT_REQUIRE_SERVER_IDENTITY)
--require-client-identity
Require clients to provide signed Ed25519 identity (default: false) (modes: discovery-service) (env: ASCII_CHAT_REQUIRE_CLIENT_IDENTITY)

NETWORK

-p, --port VAL
Server port (modes: client, server, discovery-service) (env: ASCII_CHAT_PORT)
--max-clients NUM
Maximum concurrent clients (default: 9) (modes: server, discovery-service) (env: ASCII_CHAT_MAX_CLIENTS)
--reconnect-attempts NUM
Number of reconnection attempts before giving up (-1=infinite, 0=none) (default: -1) (modes: client) (env: ASCII_CHAT_RECONNECT_ATTEMPTS)
--port-forwarding
Use UPnP/NAT-PMP port mapping to open a port in your router to ascii-chat (might fail with some routers) (default: false) (modes: server, discovery-service) (env: ASCII_CHAT_PORT_FORWARDING)
--scan
Scan for servers on local network via mDNS (default: false) (modes: client) (env: ASCII_CHAT_SCAN)
--webrtc
Make calls using WebRTC p2p connections (default: false) (modes: client, server) (env: ASCII_CHAT_WEBRTC)
--no-webrtc
Disable WebRTC, use direct TCP only (default: false) (modes: client) (env: ASCII_CHAT_NO_WEBRTC)
--prefer-webrtc
Try WebRTC before direct TCP (default: false) (modes: client) (env: ASCII_CHAT_PREFER_WEBRTC)
--webrtc-skip-stun
Skip WebRTC+STUN stage, go straight to TURN relay (default: false) (modes: client) (env: ASCII_CHAT_WEBRTC_SKIP_STUN)
--webrtc-disable-turn
Disable WebRTC+TURN relay, use STUN only (default: false) (modes: client) (env: ASCII_CHAT_WEBRTC_DISABLE_TURN)
--stun-servers STR
Comma-separated list of WebRTC+STUN server URLs (default: stun:stun.ascii-chat.com:3478,stun:stun.l.google.com:19302) (modes: client, server, discovery-service) (env: ASCII_CHAT_STUN_SERVERS)
--turn-servers STR
Comma-separated list of WebRTC+TURN server URLs (default: turn:turn.ascii-chat.com:3478) (modes: client, server, discovery-service) (env: ASCII_CHAT_TURN_SERVERS)
--turn-username STR
Username for WebRTC+TURN authentication (default: ascii) (modes: client, server, discovery-service) (env: ASCII_CHAT_TURN_USERNAME)
--turn-credential STR
Credential/password for WebRTC+TURN authentication (default: 0aa9917b4dad1b01631e87a32b875e09) (modes: client, server, discovery-service) (env: ASCII_CHAT_TURN_CREDENTIAL)
--turn-secret STR
Shared secret for dynamic WebRTC+TURN credential generation (HMAC-SHA1) (modes: client, server, discovery-service) (env: ASCII_CHAT_TURN_SECRET)
--discovery-server STR
Discovery service endpoint (IP address or hostname). (default: discovery-service.ascii-chat.com) (modes: client, server) (env: ASCII_CHAT_DISCOVERY_SERVER)
--discovery-port NUM
Discovery service port (1-65535) (default: 27225) (modes: client, server) (env: ASCII_CHAT_DISCOVERY_PORT)
--discovery-expose-ip
Allow public IP disclosure in discovery sessions (requires confirmation) (default: false) (modes: client, server) (env: ASCII_CHAT_DISCOVERY_EXPOSE_IP)

MEDIA

-f, --file STR
Stream from media file or stdin (use '-' for stdin) (modes: client, mirror) (env: ASCII_CHAT_FILE)
-u, --url STR
Requires yt-dlp in PATH. Stream from network URL (HTTP/HTTPS/YouTube/RTSP) (modes: client, mirror) (env: ASCII_CHAT_URL)
-l, --loop
Loop media file playback (not supported for --url) (default: false) (modes: client, mirror) (env: ASCII_CHAT_LOOP)
-s, --seek VAL
Seek to timestamp before playback (format: seconds, MM:SS, or HH:MM:SS.ms) (modes: client, mirror) (env: ASCII_CHAT_SEEK)
--cookies-from-browser VAL
yt-dlp option (man yt-dlp). Browser for reading cookies from (chrome, firefox, edge, safari, brave, opera, vivaldi, whale). Use without argument to default to chrome. (modes: client, mirror) (env: ASCII_CHAT_COOKIES_FROM_BROWSER)
--no-cookies-from-browser
yt-dlp option (man yt-dlp). Explicitly disable reading cookies from browser (default: false) (modes: client, mirror) (env: ASCII_CHAT_NO_COOKIES_FROM_BROWSER)

DATABASE

--database STR
Path to SQLite database for discovery session storage (modes: discovery-service) (env: ASCII_CHAT_DATABASE)

POSITIONAL ARGUMENTS

address (optional)
Server address (optional, defaults to localhost:27224)
Examples:
  • 192.168.1.1
  • example.com:8080
bind-address (optional)
Bind address (optional, can specify 0-2 addresses for IPv4/IPv6)
Examples:
  • 0.0.0.0
  • 0.0.0.0 ::
bind-address (optional)
Bind address (optional, can specify 0-2 addresses for IPv4/IPv6)
Examples:
